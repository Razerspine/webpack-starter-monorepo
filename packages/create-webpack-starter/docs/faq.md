# FAQ

## Is this a replacement for Vite / CRA?

No.

This tool is designed for:
- static sites
- CMS integrations
- full control over markup
- Pug-based workflows

---

## Can I modify webpack config after creation?

Yes.
The generated project is fully editable.
No hidden abstractions.
